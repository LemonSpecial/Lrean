# 操作系统分类

任意程序都离不开系统，如：

- Windows
- Linux
- MacOS
- Android
- IOS







# 常见文件分类

> [!CAUTION]
>
> 电脑拿到手后首先需要<font color="red">`显示文件扩展名`</font>和<font color="red">`显示隐藏文件`</font>

- 文档：txt
- 图片：jpg、jpeg、png
- 视频：mp4、flv、ts、avi、mkv
- 办公：doc、docx、xls、xlsx、ppt、pptx
- 音频：mp3、wav、aac、flac
- 压缩包：zip、rar、7z
- 可执行程序（Windows）：exe
- 可执行程序（Linux）：o、无后缀
- 可执行程序（MacOs）：app、dmg、pkg、command
- 可执行程序（Android）：apk
- 可执行程序（IOS）：ipa







# 文件系统

> [!IMPORTANT]
>
> 每个文件都拥有路径

- 绝对路径：对于根路径（盘符）下的xxx，比如：`C:\Users\Public\Documents`
- 相对路径：相对于某个文件夹下的xxx，比如：`.\Documents\MDfiles`







# 基础CMD

Windows光标键+R 打开 运行窗口，输入`cmd`

| 命令          | 实例                                            | 解释                             |
| ------------- | ----------------------------------------------- | -------------------------------- |
| `dir`         | `dir` `dir *.txt`                               | 列出所有文件，列出指定文件       |
| `cd`          | `cd Doc` `cd ..` `cd \`                         | 切换目录，返回上一级，返回根目录 |
| `mkdir` `md`  | `mk NewFolder`                                  | 创建新目录                       |
| `rmdir` `rd`  | `rmdir /s OldFolder`                            | 强制删除所有文件                 |
| `copy`        | `copy file.txt D:\`                             | **复制**某个文件到某个绝对路径   |
| `move`        | `move file.txt D:\`                             | **移动**某个文件到某个绝对路径   |
| `del` `erase` | `del temp.txt`                                  | 强制删除指定文件                 |
| `ren`         | `ren a.txt new.txt`                             | 重命名                           |
| `clear` `cls` | `clear`                                         | 清屏                             |
| `echo`        | `echo Hi`                                       | 输出文本                         |
| `tasklist`    | `tasklist`                                      | 显示所有正在运行                 |
| `taskkill`    | `taskkill /f /im 1.exe` `taskkill /pid 1111 /f` | 以名字或PID关闭程序              |
| `shutdown`    | `shutdown /s /t 60`                             | 关机                             |
